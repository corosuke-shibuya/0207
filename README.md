# ①課題名
Deep Dive MVP - コミュニケーション壁打ちSaaS

## ②課題内容（どんな作品か）
- 相手タイプ（People）と状況入力をもとに、AIと壁打ちしながら会話戦略を詰めるWebアプリです。
- Timelineの短文ログ、Coachの壁打ち、Detailでの履歴参照、SettingsのJSONエクスポート/全削除まで実装しています。

## ⑤工夫した点・こだわった点
- 「文面生成」より「AI壁打ち」にフォーカスし、対話の往復で意思決定できる体験に寄せた点。
- 相手タイプ軸（優先価値・直接性・文量・感情/論理など）を反映して、返答のトーンや提案順を変えるようにした点。
- ログイン状態とデータ表示を分離し、未ログイン時にデータが見えないようにアクセス制御を入れた点。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 壁打ち応答が似通いやすく、文脈追従と重複抑制の品質調整に時間がかかった。
- Google OAuthの開発環境設定（テストユーザー/リダイレクトURI）の詰まりどころが多かった。
- 次回はベクトル検索による文脈参照、会話の評価指標可視化、B2B向け権限設計を実装したい。
